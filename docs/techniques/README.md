## 図の見方（共通）

各ページの盤面例は **等幅フォント**で表示されるようにしている（GitHub / VS Code のプレビュー想定）。

| 記号 | 意味 |
|------|------|
| `.` | 空マス（例の都合で、まだ書いていないマス） |
| `★` | 説明の的になるマス（ここに何が入るか、候補がどうなるか） |
| `1`〜`9` | すでに確定している数字（問題のヒントや、論理で埋まったマス） |
| `|` と `+---` | 3×3 ブロックの区切り（ナンプレの太線に相当） |

**注意**: 例は **説明用の断片**であり、全体として一意に解ける完全問題とは限らない。論理の「その一歩」だけ取り出している。
