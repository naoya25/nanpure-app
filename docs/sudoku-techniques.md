# ナンプレ解法テクニック（一覧）

このアプリでは以下のテクニックについての実行支援・学習支援を行っています。

## 難易度

DB の `level` 帯や UI の「初級/中級/上級」と並べるときの **目安**。

| 段階     | レベル | 目安                                                                                 |
| -------- | ------ | ------------------------------------------------------------------------------------ |
| **初級** | 1~20   | **メモなし**でも、ある程度自明にマスを確定させることができるテクニック               |
| **中級** | 21~50  | メモがあれば自明にマスを確定させることができるテクニックおよびメモを減らすテクニック |
| **上級** | 51~100 | **メモがあっても** subset や仮定・矛盾など、パターン認識や探索が要るテクニック       |
